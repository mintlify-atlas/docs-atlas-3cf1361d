## Overview

SVAR React Gantt is a production-ready Gantt chart component for React applications. It provides an interactive timeline with drag-and-drop editing, task dependencies, hierarchical subtasks, and flexible time scales — all with full TypeScript support and React 19 compatibility.

The component is designed to handle thousands of tasks efficiently and ships with multiple built-in themes, localization support, and a rich set of UI components including toolbar, context menu, and task editor.

## Quick example

```jsx
import { Gantt } from '@svar-ui/react-gantt';
import '@svar-ui/react-gantt/all.css';

const tasks = [
  {
    id: 1,
    text: 'Project kickoff',
    start: new Date(2024, 5, 1),
    duration: 4,
    progress: 100,
    parent: 0,
    type: 'task',
  },
  {
    id: 2,
    text: 'Design phase',
    start: new Date(2024, 5, 6),
    duration: 10,
    progress: 60,
    parent: 0,
    type: 'task',
  },
];

const links = [
  { id: 1, source: 1, target: 2, type: 'e2s' },
];

export default function App() {
  return <Gantt tasks={tasks} links={links} />;
}
```
